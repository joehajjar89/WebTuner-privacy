WebTuner Privacy Policy

WebTuner does not collect, store, or transmit personal user data.

All styling preferences and configuration profiles are stored locally in the browser.

For premium users, WebTuner communicates with a remote server only to verify license activation keys. This verification does not collect browsing history, personal data, or website content.

Payments are processed securely through Stripe. WebTuner does not store or process credit card information.

No user data is sold, shared, or transferred to third parties.

If you have questions, contact: admin@m2hlabs.com
